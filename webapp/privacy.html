プライバシーポリシー

最終更新日: 2026年3月9日

AnkleCapture（以下「本アプリ」）は、足関節角度を測定するためのツールです。本アプリは、ユーザーのプライバシーを尊重し、データの安全性を最優先に考えています。

1. 収集するデータ

本アプリは以下のデータを扱います：

  • カメラ映像・写真：足関節の角度測定のために使用します
  • 被験者ID・測定者ID：ユーザーが任意に入力する識別子です
  • 測定データ：角度値、座標点、タイムスタンプ
  • デバイスの傾き情報：水平確認のために使用します

2. データの保存場所

すべてのデータはお使いのデバイス上にのみ保存されます。

  • データはデバイスのローカルストレージ（IndexedDB）に保存されます
  • 外部サーバーへのデータ送信は一切行いません
  • クラウドサービスとの同期機能はありません

3. データの共有

  • 本アプリはデータを第三者と共有しません
  • エクスポート機能（CSV、JSON、画像）はユーザーの操作によってのみ実行されます
  • AirDrop等の共有機能はユーザーが明示的に選択した場合のみ動作します

4. カメラ・センサーの使用

  • カメラ：足関節の撮影および写真のインポートに使用します
  • モーションセンサー：デバイスの傾きを検出し、水平を確認するために使用します
  • これらのアクセスはユーザーの許可が必要であり、いつでもデバイスの設定から無効化できます

5. 分析・トラッキング

本アプリはアナリティクス、広告、トラッキングツールを一切使用しません。

6. データの削除

  • アプリ内の「履歴」画面からいつでも測定データを削除できます
  • アプリを削除すると、すべてのデータが完全に消去されます

7. 医療データに関する注意

本アプリで記録されるデータには身体の測定情報が含まれる可能性があります。データの安全管理はユーザーの責任において行ってください。デバイスのパスコード設定やFace ID/Touch IDの利用を推奨します。

Privacy Policy

Last updated: March 9, 2026

AnkleCapture ("the App") is a tool for measuring ankle joint angles. The App respects user privacy and prioritizes data security.

1. Data Collected

The App handles the following data:

  • Camera images/photos: Used for ankle angle measurement
  • Subject ID / Operator ID: User-defined identifiers entered voluntarily
  • Measurement data: Angle values, coordinate points, timestamps
  • Device orientation: Used for level/horizontal verification

2. Data Storage

All data is stored exclusively on your device.

  • Data is saved in local storage (IndexedDB) on your device
  • No data is transmitted to external servers
  • There is no cloud synchronization feature

3. Data Sharing

  • The App does not share data with third parties
  • Export features (CSV, JSON, images) are initiated only by user action
  • Sharing via AirDrop or similar services only occurs when explicitly chosen by the user

4. Camera & Sensor Usage

  • Camera: Used for capturing ankle photos and importing existing photos
  • Motion sensors: Used for detecting device tilt to verify horizontal alignment
  • These permissions require user consent and can be revoked at any time in device settings

5. Analytics & Tracking

The App does not use any analytics, advertising, or tracking tools.

6. Data Deletion

  • Measurement data can be deleted at any time from the "History" screen within the App
  • Uninstalling the App permanently removes all stored data

7. Medical Data Notice

Data recorded by this App may include body measurement information. Users are responsible for ensuring data security on their devices. We recommend enabling device passcode, Face ID, or Touch ID.
